# webtoon

Naver webtoon 페이지를 보여주는 앱

## 개요
### 기능
  - 해당 요일의 네이버 웹툰 리스트를 화면에 보여줌
  - 웹툰 클릭 시 
    - 좋아요 
    - 설명, 장르, 연령, 최신 회차 (10개까지) 보여줌
    - 회차 클릭 시 해당 회차의 네이버 웹툰 페이지를 열어 화면에 보여줌
### 학습 자료와 차이점
  - 웹툰 회차를 화면에 표시할 때
    - ListView를 사용함 (학습 자료에서는 Column으로 구현)
